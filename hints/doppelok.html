All we have to do is check whether the words we received by the set generator are in the same order as in the list of words. The sorted() function and the str.index() method will help us. This str.index() method returns the index of the first element that matched the given element in the parentheses. But we will use this method as the value of the "key" argument in the sorted() function. Look carefully =) 
text = "hi world im here hi world world im here"
words = ["world", "here"]
result = {i for i in text.split() if i in words}
print(sorted(result, key=text.index))

# output
['world', 'here']
The sorted() function turned our set into a list and sorted it by text.index('world') ... text.index('here'). That is, the output is a list with the order of words, as in the specified line. Thus, we solved the problem of unordered set data type.

def words_order(text: str, words: list) -> bool:

Okay, okay! That's the spirit!=) Here is an example of a solution through for loops. The logic is very similar to our solution, but instead of the set, we check whether our text_new list contains specific words or not. 
def words_order(text: str, words: list) -> bool:
    if len(set(words)) < len(words):
        return False
    text_new = []
    for i in text.split():
        if i in words and i not in text_new:
            text_new.append(i)
    return text_new == words
