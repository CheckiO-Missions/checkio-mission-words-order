我们所要做的就是检查通过集合生成器得到的单词是否与单词表中的顺序一致。sorted() 函数和str.index() 方法将帮助我们。str.index() 方法返回与括号中给定元素匹配的第一个元素的索引。但我们将使用该方法作为 sorted() 函数中key 参数的值。仔细看 =) 

text = "hi world im here hi world world im here"
words = ["world", "here"]
result = {i for i in text.split() if i in words}
print(sorted(result, key=text.index))

输出：

['world', 'here']

sorted() 函数将我们的集合转换成了一个列表，并按照text.index('world')... text.index('here') 进行了排序。也就是说，输出是一个按指定行中的单词顺序排列的列表。因此，我们解决了无序集合数据类型的问题。

def words_order(text: str, words: list) -> bool:

    result = {i for i in text.split() if i in words}

    return sorted(result, key=text.index) == words

def words_order(text: str, words: list) -> bool:

    if len(set(words)) < len(words):
        return False
    text_new = []
    for i in text.split():
        if i in words and i not in text_new:
            text_new.append(i)
            
    return text_new == words
