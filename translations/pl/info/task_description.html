Mają Państwo tekst i sekwencję słów. Proszę sprawdzić, czy słowa w sekwencji występują w tej samej kolejności, co w podanym tekście.

Przypadki, których należy się spodziewać podczas rozwiązywania tego zadania:

  • słowo z sekwencji nie występuje w tekście - funkcja powinna zwrócić {% if is_js %}f{% else %}F{% endif %}alse;
  • dowolne słowo może pojawić się w tekście więcej niż raz - proszę użyć tylko pierwszego;
  • dwa wyrazy w podanym ciągu są takie same - funkcja powinna zwrócić {% if is_js %}f{% else %}F{% endif %}alse;
  • warunek uwzględnia wielkość liter, co oznacza, że 'hi' i 'Hi' to dwa różne słowa;
  • tekst zawiera tylko angielskie litery i spacje.
