You have a text and list of words. You need to check if words in a list have the same order as an order of their apparance in a give text.

Cases you should expect while you solving this challenge

  • word from the list is not in the text - your function should return false
  • any of words can be more than once in a text - use only the first one
    • if two words are the same - your function should return false
  • condition is case sencetive. It mease 'hi' and 'Hi' are two different words

Input: Two arguments. First is a given text, second is a list of words.

Output: Bool.
